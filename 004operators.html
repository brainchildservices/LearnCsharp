C#

C# is an object-oriented programming language created by Microsoft that runs on the .NET Framework.

C# OPERATORS

C# operators are the symbols which are used for different operations like mathematical expressions, function call

DIFFERENT KIND OF OPERATORS

  • Arithmetic operators
  • Assignment operators
  • Conditional operators
  • Unary operators
  • Increment & Decrement operators
  • Relational operators
  • Logical operators

Arithmetic Operator

Arithmetic operator is used for mathematical operations.There are 5 types of arithmetic operators.

  • +
  • -
  • *
  • /
  • %

  • : This (+) operator is called additional operator.
    It is used for the addition of two pieces of data.
    • Eg : z=x+y;
  • : This (-) operator is called subtraction operator.
    It is used for the difference of tw pieces of data.
    • Eg : z=x-y;
  • : This (*) operator is called multiplication operator.
    It is used for product of two data.
    • Eg : z=x*y;
  • : This operator is called division operator.
    It is used for the division of two data.
    • Eg : z=x/y;
  • : This operator is called modulus operator.
    It is used for finding the remainder after dividing one data from another.
    • Eg : z=x%y;

Comparison Operator

Comparison operators are used to compare two values

  • ==
  • !=
  • >
  • <
  • >=
  • <=

  • This (==) operator is called Equal to
    Eg : x==y
  • This (!=) operator is called Not equal
    Eg : x!=y
  • This (>) operator is called Greater than
    Eg : x > y
  • This (<) operator is called Greater than
    Eg : x < y
  • This (>=) operator is called Greater than or equal to
    Eg : x >= y
  • This (<=) operator is calledLess than or equal to
    Eg : x <= y

Logical Operators

Logical operators are used to determine the logic between variables or values

  • &&
  • ||
  • !

  • This (&&) operator is called Logical and
    Eg : x < 5 && x < 10
  • This (||) operator is called Logical or
    Eg : x < 5 || x < 4
  • This (||) operator is called Logical not
    Eg : !(x < 5 && x < 10)

Assignment operator

“=” is called the assignment operator.
“=” assigns the variable in the left of it with the value on the right of it.
In other words, it provides the data on its R.H.S. to the variable in its L.H.S.
