C# VARIABLES

* Variables are containers for storing data values.
* A variable is a name given to a storage area that is used to store values of various data types.

sl no Data Type size Description
1 int 4 bytes Stores whole numbers from -2,147,483,648 to 2,147,483,647
2 double 8 bytes Stores fractional numbers. Sufficient for storing 15 decimal digits
3 char 2 bytes Stores a single character/letter, surrounded by single quotes
4 string 2 bytes per character Stores a sequence of characters, surrounded by double quotes
5 bool 1 bit Stores true or false values

Different types of Variables

Integer (int)

stores integers (whole numbers), without decimals, such as 123 or -123

Correct Syntax

using System;
            
public class Program
  {
     public static void Main()
       {
          int myNum;//Declaration of an integer.
          myNum = 15;//Assignment value to the integer myNum.
          Console.WriteLine(myNum);//Accessing an integer.
        }
  }
//Output 15

Integer (int)

Wrong Syntax

            using System;
            
            public class Program
            {
            public static void Main()
              {
                integer myNum;//Wrong Declaration of integer
                int a = 5;

                bool test = true;

                int c = test; // Error. cannot convert type 'int' to 'bool'.
                
                myNum = 15 //Wrong Assignment value to the Variable myNum
                Console.WriteLine(15);//wrong Accessing of an integer.
              }
            }

Different types of Variables

Double (D)

stores floating point numbers, with decimals, such as 19.99 or -19.99

Correct Syntax

 using System;
            
 public class Program
     {
        public static void Main()
          {
              double myNum;//Declaration of double.
               myNum = 15.857D;//Assignment value to the double myNum.
               Console.WriteLine(myNum);//Accessing double.
           }
      }
//Output 15.857

Double (D)

Wrong Syntax

using System;
            
 public class Program
     {
        public static void Main()
          {
              d myNum;//Wrong Declaration of double.
               myNum = 15//WrongAssignment value to the double myNum.
               Console.WriteLine(15);//Wrong Accessing double.
           }
      }

Different types of Variables

Character (char)

stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes

Correct Syntax

using System;
            
 public class Program
  {
    public static void Main()
     {
        char myGrade;//Declaration of Char
        myGrade = 'A';//Assignment value to the Char myGrade
        Console.WriteLine(myGrade);//Accessing Char.
     }
   }
//Output A

Character (char)

Wrong Syntax

using System;
            
public class Program
   {
      public static void Main()
         {
            character myGrade;//Wrong Declaration of Char
            myGrade = "Axe"//Wrong Assignment value to the Char myGrade
            Console.WriteLine(Axe);//Wrong Accessing Char.
         }
   }

Different types of Variables

String (string)

stores text, such as "Hello World". String values are surrounded by double quotes 


using System;
            
public class Program
 {
   public static void Main()
      {
         string myName;//Declaration of string
         myName = "JOHN";//Assignment value to the string myName
         Console.WriteLine(myName);//Accessing string.
     }
  }
//Output JOHN

String (string)

Wrong Syntax

using System;
            
public class Program
   {
      public static void Main()
         {
          str myName;//Wrong Declaration of string
          myName = JOHN;//Wrong Assignment value to the string myName
          myName = 'JOHN'//Wrong Assignment value to the string myName
          Console.WriteLine("JOHN");//Wrong Accessing string.
         }
   }

Different types of Variables

Boolean (bool)

stores values with two states: true or false 


using System;

namespace MyApplication
   {
      class Program
        {
            static void Main(string[] args)
              {
                    bool isCSharpFun = true;
                    bool isFishTasty = false;
                    Console.WriteLine(isCSharpFun);   // Outputs True
                    Console.WriteLine(isFishTasty);   // Outputs False    
              }
        }
    }
