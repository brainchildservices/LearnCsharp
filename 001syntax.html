What is Console Application

A Console Application in C# context is an application that takes inputs and displays output at a command-line console with access to three basic streams:

  • standard input
  • standard output
  • standard error

Why to Use

  1. To provide a simple user interface for applications requiring little or no user interaction.
    • samples for learning C# language features
    • command-line utility programs
  2. Automated testing, which can reuse automation implementation resources.
    • Instead of Graphical Interface console application have Character based interfaces.
