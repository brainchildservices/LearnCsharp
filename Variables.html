C# VARIABLES

* Variables are containers for storing data values.
* A variable is a name given to a storage area that is used to store values of various data types.

Different types of Variables

Integer (int)

stores integers (whole numbers), without decimals, such as 123 or -123 



            using System;
            
            public class Program
            {
            public static void Main()
              {
                int myNum;//Declaration of variable
                myNum = 15;//Assignment value to the Variable myNum
                Console.WriteLine(myNum);
              }
            }
			//Output 15

Different types of Variables

Double (D)

stores floating point numbers, with decimals, such as 19.99 or -19.99 


            using System;
            
            public class Program
            {
            public static void Main()
              {
                double myNum;//Declaration of variable
                myNum = 15.857D;//Assignment value to the Variable myNum
                Console.WriteLine(myNum);
              }
            }
			//Output 15.857

Different types of Variables

Character (char)

stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes 


            using System;
            
            public class Program
            {
            public static void Main()
              {
                char myGrade;//Declaration of variable
                myGrade = 'A';//Assignment value to the Variable myGrade
                Console.WriteLine(myGrade);
              }
            }
			     //Output A

Different types of Variables

String (string)

stores text, such as "Hello World". String values are surrounded by double quotes 


            using System;
            
            public class Program
            {
            public static void Main()
              {
                string myName;//Declaration of variable
                myName = "JOHN";//Assignment value to the Variable myName
                Console.WriteLine(myName);
              }
            }
			//Output JOHN
